# spell-it #
Small Node.js module that writes out numbers as words in English, French, Indonesian or Danish.

## Usage ##
Install:
```
npm install spell-it
```
From node.js:
```js
var spell = require('spell-it') ('fr');
console.log(spell(42));
// quarante-deux
```
From the shell:
```
$ ./cli.js 42 en
```
